Criamos um script seguro, que a senha não fica inserido no código fonte, ele esta separado em um arquivo JS, (destaque_hpg.js) este nome destaque hpg é uma forma de disfarce para este script, assim o internauta que for ver a pasta Temporary internet files não perceberá que neste arquivo vai conter as senhas, em caso de mudança do nome do arquivo é necessário mudar no código fonte da página.

A tag que é nessesário a mudança em caso de alteração do arquivo js é:

<script src="destaque_hpg.js"></script>

Não esqueça de alterar a senha e as paginas que devem ser abertas, abra o arquivo destque_hpg.js no Bloco de Notas, não clique duas vezes sobre este arquivo, abra o programa bloco de notas e abra o arquivo, em seguida edite e salve.

Simples!!!!
